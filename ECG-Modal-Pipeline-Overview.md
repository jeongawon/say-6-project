## 4. 신호 전처리 파이프라인

### 전체 흐름

```
┌──────────────┐     ┌──────────────┐     ┌──────────────┐
│  .hea + .dat │     │  NaN 보간    │     │  리샘플링     │
│  (wfdb 읽기) │ ──→ │  + 클리핑    │ ──→ │  500→100Hz   │
│  (5000, 12)  │     │  ±3mV       │     │  (1000, 12)  │
└──────────────┘     └──────────────┘     └──────┬───────┘
                                                  │
┌──────────────┐     ┌──────────────┐             │
│  npy 저장    │     │  12채널 정렬  │             │
│  ~48KB/file  │ ←── │  채널명 매핑  │ ←───────────┘
└──────────────┘     └──────────────┘
```

### 각 단계 상세

| 단계 | 처리 | 입력 | 출력 | 비고 |
|------|------|------|------|------|
| 1 | 파형 읽기 | .hea + .dat | (5000, 12) | wfdb 라이브러리 |
| 2 | NaN 보간 | 결측 샘플 | 보간된 신호 | pandas 선형 보간 |
| 3 | 클리핑 | 원본 진폭 | ±3mV 제한 | 극단 노이즈 제거 |
| 4 | 리샘플링 | 500Hz | 100Hz | resampy (고품질) |
| 5 | 채널 정렬 | 가변 순서 | 고정 12채널 | 채널명→인덱스 매핑 |
| 6 | 저장 | numpy 배열 | .npy 파일 | float32, ~48KB |

### ⚠️ 적용하지 않는 것들 (ECG-MIMIC 논문과 동일)

```
❌ 밴드패스 필터 (0.5-40Hz)    → 모델이 학습으로 대체
❌ 60Hz 노치 필터              → 모델이 학습으로 대체
❌ 베이스라인 원더링 보정       → 모델이 학습으로 대체
❌ R-peak 검출                 → 사용 안 함
❌ 데이터 증강                 → 사용 안 함
```

### 정규화 (학습 시 적용)

## 5. 인구통계 결합 (논문 대비 차별화 ①)

ECG-MIMIC 논문은 ECG 신호만 입력. 우리는 나이/성별을 추가 입력으로 결합.

### 근거

같은 ECG 파형이라도 환자에 따라 임상적 의미가 다름:

```
심방세동 유병률:   50대 ██ 2%     →    80대 ███████████████ 15%
급성 심근경색:     여성 ████ 1배   →    남성 ████████ 2배
고칼륨혈증:        젊은층 █ 낮음   →    고령+신부전 ████████ 높음
```

### 전처리

| 피처 | 원본 | 변환 | 예시 |
|------|------|------|------|
| 나이 | 18~101세 | (age-18)/(101-18) → 0~1 | 60세 → 0.51 |
| 성별 | M / F / missing | 1.0 / 0.0 / 0.5 | 여성 → 0.0 |

### 모델 내 결합 구조

```
ECG 신호 (1000, 12)                    나이/성별 (2)
       │                                    │
  S4/Transformer                        FC (2→16)
       │                                    │
  ECG 임베딩 (512)                     demo 임베딩 (16)
       │                                    │
       └──────────── concat ─────────────────┘
                       │
                  (528차원)
                       │
                  FC → 24개 예측
```

## 6. 긴급도 가중 Loss (논문 대비 차별화 ②)

### 핵심 아이디어

```
기존 (논문):  Loss = BCE(pred, target)           ← 모든 질환 동등
우리:        Loss = Σ weight[i] × BCE(pred[i])   ← 사망률 기반 가중
```

### 🔍 핵심 발견

> 2차 타겟(비심혈관)인 **패혈증(28.1%)**, **호흡부전(27.5%)**, **급성 신부전(12.6%)**이
> 1차 타겟(심혈관)인 심방세동(8.8%)보다 사망률이 **3배 이상** 높음
>
> → "ECG만으로 비심혈관 고위험 질환을 조기 감지할 수 있다"는 것이 강력한 기여점

### 긴급도 등급 체계

| Tier | 기준 | 가중치 | 질환 수 | 의미 |
|------|------|--------|--------|------|
| Tier 1 | 사망률 ≥ 10% | 3.0 | 10개 | 놓치면 사망 |
| Tier 2 | 사망률 5~10% | 2.0 | 11개 | 빠른 감지 필요 |
| Tier 3 | 사망률 2~5% | 1.5 | 3개 | 참고 정보 |

## 8. 평가 체계

| 지표 | Tier 1 (≥10%) | Tier 2 (5~10%) | Tier 3 (2~5%) |
|------|:------------:|:--------------:|:-------------:|
| AUROC | ≥ 0.80 | ≥ 0.75 | ≥ 0.65 |
| Recall@0.5 | ≥ 0.90 | ≥ 0.80 | ≥ 0.70 |
