# 흉부 X-Ray 14개 질환 — 실제 방사선과 전문의 판독 기준

> 출처: Radiology Assistant (Netherlands), Radiopaedia, StatPearls (NCBI), RadiologyMasterclass (UK NHS),
> Radiology Key, RSNA RadioGraphics, Medscape eMedicine, Osmosis, PMC 논문 다수
> 이 문서는 실제 영상의학과 전문의가 PA 흉부 X-Ray에서 각 질환을 어떻게 판독하는지를 정리한 것입니다.

---

## 판독의 기본 원칙

실제 흉부과 전문의는 "체계적 접근법(systematic approach)"으로 판독합니다.
안쪽에서 바깥쪽으로(inside-out) 순서대로 봅니다:

1. 기도(Airway) — 기관 위치, 기관지
2. 심장/종격동(Heart/Mediastinum) — 심장 크기, 대동맥, 종격동 윤곽
3. 폐야(Lungs) — 양쪽 폐 상/중/하엽, 폐문부
4. 흉막(Pleura) — 늑골횡격막각, 흉막선
5. 뼈/연조직(Bones/Soft tissue) — 늑골, 쇄골, 척추
6. 의료기구(Devices) — 튜브, 카테터 위치

각 질환마다 "어디를", "무엇을", "어떤 징후(sign)로" 보는지가 명확히 다릅니다.

---

## 1. Atelectasis (무기폐) — "폐가 쪼그라든 곳"

### 전문의가 보는 것
- **폐 용적 감소**: 해당 폐엽이 줄어들면서 주변 구조물이 끌려옴
- **종격동 이동(mediastinal shift)**: 무기폐 쪽으로 기관/심장이 끌려감
- **횡격막 거상**: 해당 쪽 횡격막이 위로 올라감
- **늑골 간격 좁아짐**: 해당 쪽 늑골 사이가 좁아짐
- **보상성 과팽창**: 정상 폐가 과팽창되어 더 투명해짐

### 위치별 특이 소견
- 우상엽 무기폐: 우측 기관지가 위로 끌려가고, 소열이 위로 이동
- 좌하엽 무기폐: 좌측 횡격막 뒤쪽으로 삼각형 음영 (Sail sign)
- 판상 무기폐(plate-like): 수평의 얇은 선상 음영 (주로 폐 하부)

### 핵심 징후(Sign)
- Golden S sign: 우상엽 무기폐 + 폐문부 종괴가 있을 때 S자 곡선
- Silhouette sign: 무기폐 위치에 따라 심장 윤곽이나 횡격막 윤곽이 소실

### Grad-CAM이 못 잡는 이유
무기폐는 "없어진 것(용적 감소)"을 보는 것인데, AI는 "있는 것(밝은 음영)"을 찾음.
횡격막 거상, 종격동 이동 같은 간접 징후는 분류 모델이 학습하기 어려움.

---

## 2. Cardiomegaly (심비대) — "심장이 너무 큰 것"

### 전문의가 보는 것
- **심흉비(Cardiothoracic Ratio, CTR)**: 심장 가로 폭 / 흉곽 가로 폭
- CTR > 0.5 (50%)면 심비대 진단
- **자(ruler)로 실제 측정**합니다 — 정량적 판단

### 세부 측정
- 심장 우측 경계: 우심방 — 우측으로 돌출 정도
- 심장 좌측 경계: 좌심실 — 좌측 아래로 돌출 정도
- 좌심실 비대: 좌측 하방으로 심장 첨부(apex)가 내려감
- 우심실 비대: 심장 첨부가 위쪽으로 들림 (boot-shaped heart)
- 좌심방 확대: 좌측 기관지가 위로 밀림, 심장 뒤쪽 이중 윤곽

### 주의
- 반드시 PA 뷰에서만 측정 (AP 뷰에서는 심장이 확대되어 보임)
- 비만, 임신, 복수 → 횡격막 거상 → 심장이 가로로 누워서 CTR 거짓 양성

### AI가 이걸 잘 잡는 이유
CTR은 심장 윤곽 크기 비율이라 CNN이 상대적으로 잘 학습함.
Claude Code 분석에서도 Cardiomegaly 히트맵이 심장에 집중된 건 이 때문.

---

## 3. Consolidation (경화) — "폐포가 액체로 찬 곳"

### 전문의가 보는 것
- **균일한 음영 증가(homogeneous opacity)**: 해당 폐엽이 뿌옇게 하얘짐
- **공기기관지조영(air bronchogram)**: 주변 폐포는 물로 찼는데 기관지는 공기가 남아서 검은 가지처럼 보임 — 경화의 결정적 징후
- **Silhouette sign**: 경화가 심장이나 횡격막에 붙어있으면 윤곽이 소실됨
  - 심장 좌측 경계 소실 → 좌상엽 설엽(lingula) 경화
  - 심장 우측 경계 소실 → 우중엽 경화
  - 좌측 횡격막 소실 → 좌하엽 경화

### 위치 파악
- Silhouette sign으로 앞쪽/뒤쪽을 구분
- 폐엽 경계를 따라 쐐기형(wedge-shaped) 음영이면 대엽성 폐렴

### Pneumonia와의 관계
경화는 "영상 소견"이고 폐렴은 "임상 진단"임. 열 + 기침 + 경화 = 폐렴으로 진단.
같은 영상 소견이라도 임상 정보에 따라 폐렴/폐출혈/폐부종 등으로 달라짐.

---

## 4. Edema (폐부종) — "폐에 물이 찬 것"

### 전문의가 보는 것 — 4단계 진행
1. **폐정맥 확장(cephalization)**: 정상은 하엽 혈관이 굵은데, 폐부종 초기에는 상엽 혈관이 하엽보다 굵어짐
2. **간질 부종(interstitial edema)**: 폐문 주위 혈관 윤곽이 뿌옇게 흐려짐, Kerley B 선 (늑골횡격막각 근처의 짧은 수평선)
3. **폐포 부종(alveolar edema)**: 양쪽 폐문 주위로 대칭적인 뿌연 음영 — "나비날개(butterfly/bat wing) 패턴"
4. **흉수(pleural effusion)**: 양쪽 늑골횡격막각이 뭉뚝해짐

### 핵심 감별
- 심부전 폐부종: 양측 대칭, 심비대 동반, 상엽 혈관 확장
- ARDS 폐부종: 비대칭 가능, 심장 크기 정상, 경과가 빠름

### AI가 어려운 이유
초기 cephalization은 매우 미세한 혈관 굵기 차이 → 224x224에서는 보이지 않음.
진행된 폐부종은 Pneumonia/Consolidation과 영상 소견이 거의 동일 → AI가 혼동.

---

## 5. Enlarged Cardiomediastinum (종격동 확대) — "가운데가 넓어진 것"

### 전문의가 보는 것
- **종격동 너비**: 기관 양쪽의 종격동이 8cm 이상이면 확대
- **대동맥 윤곽**: 대동맥궁(aortic knob)이 커지거나 석회화
- **기관 편위**: 종괴에 의해 기관이 한쪽으로 밀림
- **부대동맥선(paratracheal stripe)**: 정상 3mm 이하, 두꺼워지면 림프절 비대

### 원인별 차이
- 대동맥 확장: 대동맥궁 돌출, 석회화 동반 가능
- 림프절 비대: 기관 주위, 기관 분기부 확대
- 대동맥 박리: 급성이면 종격동이 갑자기 넓어짐 — 응급

### 희귀 질환 (양성 3.36%)
대부분의 CXR에서 정상이라 양성 샘플이 극히 적음. AI 학습이 가장 어려운 질환 중 하나.

---

## 6. Fracture (골절) — "뼈가 부러진 곳"

### 전문의가 보는 것
- **늑골 골절**: 각 늑골을 하나씩 따라가며 피질(cortex) 불연속 확인
- **쇄골 골절**: 쇄골 양쪽 끝 확인
- **척추 골절**: 척추체 높이 감소, 쐐기형 변형
- **견갑골 골절**: 드물지만 외상 환자에서 확인

### 주의
- 늑골 골절은 CXR에서 30~50%가 놓침 — CT가 훨씬 정확
- 특히 앞쪽 늑골, 연골 부위는 CXR에서 잘 안 보임
- 전문의도 놓치기 쉬운 질환 → AI도 어려움

### AI가 못 잡는 이유
골절선은 1~2 픽셀 수준의 극히 미세한 불연속 → 224x224 해상도에서는 거의 보이지 않음.
원본 해상도(2000x2000)에서도 어렵기 때문에 DenseNet-121으로는 한계가 명확.

---

## 7. Lung Lesion (폐 병변/결절) — "동그란 것이 보임"

### 전문의가 보는 것
- **결절(nodule)**: 3cm 미만의 둥근 음영
- **종괴(mass)**: 3cm 이상의 둥근 음영 — 악성 가능성 높음
- **경계(margin)**: 매끈하면 양성 가능성, 불규칙/침상(spiculated)이면 악성 의심
- **석회화(calcification)**: 중심부 석회화면 양성, 편심 석회화면 악성 의심
- **공동(cavitation)**: 내부가 비어있으면 결핵/농양/괴사성 종양

### 위치 확인
- 모든 폐야를 구석구석 확인 (폐 첨부, 쇄골 뒤, 심장 뒤, 횡격막 아래)
- 이전 영상과 비교하여 크기 변화 추적 (doubling time)

### AI가 어려운 이유
작은 결절은 혈관과 겹쳐서 구분이 어려움. CT에서는 잘 보이지만 CXR에서는 1cm 이하 결절의 민감도가 매우 낮음.

---

## 8. Lung Opacity (폐 음영) — "뭔가 뿌연 것"

### 전문의가 보는 것
이건 사실 **가장 비특이적인(non-specific) 소견**임. "폐에 뭔가 있다"는 뜻.

- **위치**: 어떤 폐엽에 있는지
- **형태**: 균일(homogeneous) vs 불균일(heterogeneous)
- **경계**: 명확 vs 불명확
- **분포**: 한쪽 vs 양쪽, 중심부 vs 주변부
- **동반 소견**: 공기기관지조영, 흉수, 심비대 등

### 감별
같은 "폐 음영"이라도 동반 소견으로 다르게 해석:
- 음영 + 공기기관지조영 + 열 → 폐렴
- 음영 + 심비대 + 양측 대칭 → 폐부종
- 음영 + 폐용적 감소 → 무기폐
- 음영 + 둥근 형태 → 종괴

### AI에서의 의미
Lung Opacity는 다른 질환(Consolidation, Pneumonia, Edema, Atelectasis)과 중복.
하나가 양성이면 Lung Opacity도 거의 자동으로 양성 → 확률이 같이 올라가는 이유.

---

## 9. No Finding (소견 없음) — "정상"

### 전문의가 보는 것
- 위의 모든 항목을 체계적으로 확인한 후 **아무 이상이 없을 때**
- "정상"이라고 판독하는 것이 가장 어려움 — 미세한 이상을 놓치지 않았는지 확인 필요

### AI에서의 의미
No Finding은 나머지 13개가 전부 음성일 때만 양성.
CheXpert labeler에서는 특정 키워드("hyperinflation" 등)가 있으면 No Finding도 음성 처리.

---

## 10. Pleural Effusion (흉수) — "흉막에 물이 고인 것"

### 전문의가 보는 것
- **늑골횡격막각(costophrenic angle) 둔화**: 정상은 날카로운 각도, 흉수가 고이면 뭉뚝해짐 — 가장 첫 번째 징후 (약 200~300mL부터)
- **메니스커스(meniscus sign)**: 흉수가 올라가면서 바깥쪽이 높고 안쪽이 낮은 오목한 곡선
- **횡격막 소실**: 흉수가 많아지면 횡격막 윤곽이 안 보임
- **반투명 음영(veil-like opacity)**: 한쪽 폐 전체가 뿌연 안개처럼 → 혈관은 여전히 보임 (경화와 구분점)

### 양 추정
- 늑골횡격막각만 둔화: 소량 (~300mL)
- 횡격막 절반 이상 가림: 중등량 (~500-1000mL)
- 한쪽 폐야 전체 음영 + 종격동 반대쪽 이동: 대량 (>1000mL)

### 중요 감별
- 흉수 vs 무기폐: 둘 다 한쪽이 뿌옇게 되지만, 흉수는 종격동이 반대쪽으로 밀리고 무기폐는 같은 쪽으로 끌려감

---

## 11. Pleural Other (기타 흉막 이상) — "흉막의 다른 문제"

### 전문의가 보는 것
- **흉막 비후(pleural thickening)**: 늑골 안쪽에 얇은 줄 음영
- **흉막 석회화(pleural calcification)**: 석면 노출 이력 있으면 홀리잎(holly leaf) 모양
- **흉막 종괴**: 불규칙한 결절상 비후 → 악성(중피종) 의심

### 극히 드묾 (양성 1.57%)
AI 학습에 양성 샘플이 너무 적어 신뢰도 낮음. pos_weight 63.76으로 가장 높은 불균형.

---

## 12. Pneumonia (폐렴) — "폐에 감염"

### 전문의가 보는 것
- 폐렴은 **영상 소견 + 임상 정보** 종합 판단
- 영상만으로는 경화(Consolidation)와 구분 불가
- 열 + 기침 + 백혈구 상승 + 경화 = 폐렴

### 패턴별 분류
- **대엽성 폐렴**: 한 폐엽 전체를 채우는 균일한 경화, 공기기관지조영
- **기관지 폐렴**: 여러 곳에 패치(patch) 형태로 산재
- **간질성 폐렴**: 미만성 그물(reticular) 패턴, 폐야 전체

### AI가 어려운 이유
"폐렴"은 영상이 아니라 임상 진단. 같은 경화 소견이 폐렴일 수도 있고 폐출혈, 폐부종일 수도 있음.
CheXpert labeler는 보고서에 "pneumonia"라는 단어가 있으면 양성으로 분류 → 영상 소견과 불일치 가능.
U-Ones 후 양성이 6.18% → 12.47%로 뛴 것도 "possible pneumonia"가 전부 양성이 되어서.

---

## 13. Pneumothorax (기흉) — "폐와 흉벽 사이에 공기"

### 전문의가 보는 것
- **장측 흉막선(visceral pleural line)**: 폐 가장자리에 가느다란 흰 선이 보이고, 그 바깥에는 폐 문양(lung marking)이 없음
- **크기 측정**: 폐 가장자리에서 흉벽까지 거리가 2cm 이상이면 "대량(large)"
- **긴장성 기흉(tension pneumothorax)**: 종격동이 반대쪽으로 밀림 + 횡격막 하강 → 응급!

### 놓치기 쉬운 상황
- 앙와위(supine) 촬영: 기흉이 앞쪽으로 모여서 폐첨부에 안 보임 → "deep sulcus sign" (늑골횡격막각이 비정상적으로 깊어짐)으로 판단
- 작은 기흉: 호기(expiration) 촬영으로 더 잘 보임

### AI가 비교적 잘 잡는 질환
폐 문양이 갑자기 사라지는 것은 CNN이 학습하기 상대적으로 쉬움.
논문에서도 Pneumothorax AUROC가 0.94~0.95로 가장 높은 편.

---

## 14. Support Devices (의료 기구) — "튜브와 선이 보이는 것"

### 전문의가 보는 것
- **기관내관(ETT)**: 기관 분기부(carina) 3~5cm 위에 팁이 위치해야 정상
- **중심정맥관(CVC)**: 상대정맥/우심방 접합부에 팁이 위치해야 정상
- **비위관(NG tube)**: 위장 내에 팁이 위치해야 정상
- **흉관(chest tube)**: 흉막강 내에 위치, 기흉이면 첨부 방향, 흉수면 하부 방향
- **심박동기(pacemaker)**: 리드선이 우심방/우심실에 위치

### 판독의 핵심
위치가 "정상인가 비정상인가"를 판단 — 잘못된 위치는 즉시 보고
ETT가 우주기관지로 깊이 들어가면 → 좌폐 무기폐 위험
CVC 팁이 너무 깊으면 → 심장 부정맥 위험

### AI가 잘 잡는 이유
금속/플라스틱 튜브는 밝은 선상 구조물로 명확하게 보여서 CNN이 쉽게 학습.
Claude Code 분석에서도 Support Devices 히트맵이 기구 위치에 정확히 집중.

## 종합: "위치를 정확히 짚는 것"이 핵심

위 14개를 보면 전문의는 거의 모든 질환에서 **"어디에"** 있는지를 먼저 확인합니다.

| 질환 | 전문의가 보는 핵심 위치 |
|------|------------------------|
| Atelectasis | 용적 감소된 특정 폐엽 + 종격동 이동 방향 |
| Cardiomegaly | 심장 윤곽 양쪽 끝 → CTR 계산 |
| Consolidation | 특정 폐엽의 균일 음영 + Silhouette sign으로 위치 확정 |
| Edema | 양쪽 폐문 주위 대칭 + 상엽 혈관 확장 |
| Enlarged Cardiomediastinum | 종격동 너비 + 대동맥궁 |
| Fracture | 각 늑골/쇄골을 하나씩 추적 |
| Lung Lesion | 폐야 구석구석 + 뼈 뒤쪽/심장 뒤쪽 숨은 영역 |
| Lung Opacity | 위치+형태+동반소견으로 원인 감별 |
| No Finding | 모든 영역 정상 확인 |
| Pleural Effusion | 늑골횡격막각 → 횡격막 → 폐야 순서 |
| Pleural Other | 흉막 표면 따라 비후/석회화 |
| Pneumonia | 경화 위치 + 임상 정보 종합 |
| Pneumothorax | 폐 가장자리의 흉막선 + 폐 문양 소실 |
| Support Devices | 각 기구 팁의 해부학적 위치 |

→ 이것이 **Grad-CAM이 부적합하고, 바운딩 박스(YOLOv8) 또는 세그멘테이션(U-Net)이 필요한 근본 이유**입니다.
분류 모델은 "있다/없다"만 답하지, "어디에"는 답하지 못합니다.
